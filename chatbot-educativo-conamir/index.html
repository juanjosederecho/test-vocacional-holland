Instrucciones para publicar en GitHub Pages
  1. Crea un repositorio publico en GitHub, por ejemplo: chatbot-educativo-conamir.
  2. Sube este archivo con el nombre exacto index.html.
  3. Entra a Settings / Ajustes del repositorio.
  4. Abre Pages.
  5. En Source / Fuente, elige desplegar desde la rama principal y carpeta raiz.
  6. Guarda y espera unos segundos a que GitHub publique el enlace.
  7. Abre el enlace, toca Configurar, pega tu API Key y empieza a usar el chatbot.
